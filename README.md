# stickies.el — sticky notes in dedicated frames

Pastel-colored panes of text that auto-save, floating above other application windows.
Creating a new note doesn't ask for a file name, a commitment-free writing space.
Notes can collapse into a title-bar to hide their content.

Heavily inspired by Apple's Stickies.app, this package provides a similar experience, but
you can use any Emacs mode for the content. The notes are just buffers stored in text
files.
